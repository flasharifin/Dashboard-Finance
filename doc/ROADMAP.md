
---
High Impact — Langsung Bantu Keputusan

1. Target Alokasi & Rebalancing Calculator

Pengguna set target % per aset class (IDX 60%, US 30%, Crypto 10%). App hitung drift dari target dan tampilkan berapa rupiah harus beli/jual tiap aset untuk kembali ke target. Ini yang paling sering dicari investor saat mau rebalancing.

2. Watchlist & Pipeline Investasi

Aset yang sedang diobservasi sebelum masuk — set target harga beli, notasi alasan, status (observasi/siap beli/skip). Jadi "antrian" investasi berikutnya yang terstruktur.

3. Goal Tracker

Input: target nominal (Rp 1 M) + deadline (2030) → app hitung berapa harus diinvestasikan per bulan, proyeksi dengan asumsi return X%, dan progress saat ini vs target.

---
Medium Impact — Analisis Lebih Dalam

4. Risk Dashboard per Posisi (DONE)

Volatilitas (std dev 30 hari), max drawdown, dan konsentrasi risiko. Kalau 1 saham dominasi 40% portofolio, langsung terlihat.

5. Dividend Calendar

Jadwal cum-date dan payment-date dividen untuk saham yang dipegang. Bantu planning cash flow.

6. Tax Summary (SPT-ready)

Rekapitulasi realized P&L + dividen per tahun fiskal untuk keperluan pelaporan SPT. Sangat praktis di Q1.

---
Nice to Have (ALL DONE)

7. Correlation Matrix

Visual seberapa berkorelasi antar aset — bantu nilai diversifikasi nyata (bukan cuma jumlah saham).

8. Stress Test Simulator

"Kalau IHSG turun 20%, BBCA turun 15%, portfolio saya jadi berapa?" — simulasi skenario bear market.
